# Quick Start

1. Make sure Git & NodeJS are installed
2. `git clone https://github.com/syahmifauzi/microservices.git`
3. `cd microservices/<MICROSERVICE>/`
4. `npm install`
5. `npm run dev`
6. Open web browser & navigate to `http://localhost:<PORT>`
